Tags de Título(voltar para o indice)

Parágrafos e personalização de texto(voltar para o indice)

Os parágrafos, como este em que estou escrevendo, são denominados pela tag <p>...</p>

As tags de personalização de texto são utilizadas para destacar partes dos textos para que as partes necessárias sejam enfatizadas

Algumas tags de personalização são:

  • <mark>...</mark>, utilizada para simular um marca texto no site
  • <strong>...</strong>, utilizada para deixar o texto em negrito
  • <i>...</i>, utilizada para deixar o texto em italico
  • <u>...</u>, utilizada para deixar o texto sublinhado
  • <del>...</del>, utilizada para "riscar" o texto, simulando uma alteração
  • <small>...</small>, utilizada para deixar o texto menor
  • Exemplo<sub>...</sub>, utilizada para criar um texto menor na parte inferior
  • Exemplo<sup>...</sup>, utilizada para criar um texto menor na parte superior
  • <blockquote>...</blockquote>
    utilizada para adicionar citações de outras fontes
  • <hr>...</hr>, utilizada para adicionar uma linha horizontal, como essa aqui de baixo

Listas(voltar para o indice)

Estas tags são utilizadas para criar listas ordenadas(numeradas) e não ordenadas, as chamadas "bullet lists"

As tags são:

  • <ol>...</ol>, para listas ordenadas:
    1. item 1
    2. item 2
  • <ul>...</ul>, para listas não ordenadas:
    • item 1
    • item 2

A estrutura de uma lista também deve conter a tag <li>...<li> que estipula os itens das listas

No fim, a estrutura deve ficar assim:

  • Para listas ordenadas
    <ul>
        <li>...</li>
        <li>...</li>
    </ul>
  • Para listas não ordenadas
    <ol>
        <li>...</li>
        <li>...</li>
    </ol>

Outras tags(voltar para o indice)

Aqui eu trago outras tags citadas que não se encaixam exatamente nos tópicos anteriores

  • a tag <a>...</a>

    É utilizada para incluir links externos ou links numa mesma página, como o índice, que permite clicar em um link que te leva
    para uma parte da mesma página, ou os links citados nas referências.

  • a tag <abbr title="...">...</abbr>

    É utilizada para incluir a descrição de uma abreviação, como por exemplo: CRUD(passe o cursor por cima)
    Onde o title="..." recebe a descrição da abreviação.

  • Na descrição do desafio foi citada a tag <font>...</font> que era utilizada para
    formatar um texto alterando a cor, fonte e tamanho, mas não é mais utilizada, hoje em dia se utiliza o CSS para formatar os textos
    Para formatar a cor, fonte e tamanho de um parágrafo com CSS a sintaxe seria a seguinte:
    • Para alterar a cor
      <p style="color:blue">Este texto é azul</p>
      ficaria assim:

      Este texto é azul

    • Para alterar a fonte
      <p style="font-family:verdana">Este texto está com a fonte verdana</p>
      ficaria assim:

      Este texto está com a fonte verdana

    • Para alterar o tamanho
      <p style="font-size:25px">Este texto está com um tamanho de 25 pixels</p>
      ficaria assim:

      Este texto está com um tamanho de 25 pixels
